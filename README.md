## 功能

- nginx conf 编辑
- host 编辑
- node 项目列表，点击可在 vscode 中打开（需配置 code 命令）
- 本地需求文档和设计文档列表，支持点击在 finder 中查看或在浏览器直接打开
- 支持自定义配置相关目录

## start

```shell
npm install
npm start
```

## build

```shell
npm run build
```
