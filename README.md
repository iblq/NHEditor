基于 electron 实现的 Mac 下简单工作流工具。

## 功能
- nginx conf 编辑
- host 编辑
- node 项目列表，支持点击在 vscode 中打开
- 本地需求文档和设计文档列表，支持点击在 finder 中查看或 在浏览器直接打开

> host 编辑需要管理员权限，sudo 命令执行相关方法参考 SwitchHosts，感谢

## start

```shell
npm install
npm start
```

## build

```shell
npm run package
// or
npm run package2
```
